# Primeiro CRUD

<h2>Gerenciador de Tarefas</h2>
<ul>
<li>Listar todas as tarefas</li>
<li>Criar uma nova tarefa</li>
<li>Buscar tarefa pelo id</li>

<li>Excluir Tarefa</li>
</ul>

<li>Springboot</li>
<li>Swagger (Springfox)</li>
<li>H2 Database</li>
<li>PostgreSQL</li>
</ul>
